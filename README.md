# Enterprise Network Simulation (CCNA Project)

## 📌 Overview
This project simulates a real enterprise network using Cisco Packet Tracer. It connects two branches with full communication between all devices.

## 🏢 Network Design

### Branch 1
- 64 PCs  
- 12 IP Phones  
- 11 Switches  
- 1 Router  
- 1 Server  

### Branch 2
- 60 PCs  
- 8 IP Phones  
- 6 Switches  
- 1 Router  

## ⚙️ Features
- DHCP for automatic IP assignment  
- Static IP for server (192.168.1.1)  
- VoIP communication system  
- Routing between branches  
- Full network connectivity  

## 🧪 Testing
All devices are able to communicate successfully across both branches.
